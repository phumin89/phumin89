# Spotify Setup

This profile uses a custom SVG card generated by `scripts/generate_spotify_svg.py`.

## 1. Create a Spotify app

1. Open the Spotify Developer Dashboard.
2. Create an app.
3. Add this redirect URI:

```text
http://127.0.0.1:8888/callback
```

## 2. Get a refresh token

Run:

```powershell
python scripts/get_spotify_refresh_token.py
```

The script prints:

- `SPOTIFY_CLIENT_ID`
- `SPOTIFY_CLIENT_SECRET`
- `SPOTIFY_REFRESH_TOKEN`
- your Spotify `user_id`

## 3. Add GitHub repository secrets

In `phumin89/phumin89` add:

- `SPOTIFY_CLIENT_ID`
- `SPOTIFY_CLIENT_SECRET`
- `SPOTIFY_REFRESH_TOKEN`

## 4. Run the workflow

Open GitHub Actions and run `Update Spotify Card`, or wait for the schedule.
